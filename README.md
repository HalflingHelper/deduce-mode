# deduce-mode

deduce-mode exists to  moderately improve the experience of writing [deduce](https://github.com/jsiek/deduce/) in VS Cde. Inspired by agda-mode, though
much less fully featured.

## Features

- Syntax Highlighting
- File icons

More to come!
- Snippet completion
- Semantic Highlighting
